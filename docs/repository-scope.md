# Repository Scope

This showcase repository is a curated public layer, not a mirror of the private production repository.

## Scope Summary

| In scope | Out of scope |
|---|---|
| product overview | full shipping app |
| visual system examples | closed-source analyzer runtime |
| analyzer contract docs | private model assets and datasets |
| mock data and screenshots | business-sensitive logic |
| curated architecture notes | internal experiments and ops |

## Working Rule

If a file helps someone understand Voiceland without materially helping them clone the core product capability, it is a candidate for this repo.
